Exercise 5: Selectors

This exercise is about using selectors to target the DOM.

Generate a log message for each of the following steps:

  • Change the text color of the second paragraph.
  • Change the background of the first paragraph.
  • Remove the third paragraph.
  • Change the text color of the fourth list item.
  • Change the text of the second list item to say "I did it!"
