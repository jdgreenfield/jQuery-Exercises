Exercise 4: Changing the DOM.

This exercise is about adding and removing content.

Log a console message for each of the following steps:

  • Remove the text from the paragraph in the div of class 'removeMe'.
  • Remove the paragraph from the div of class 'removeMe'.
  • Remove the div of class 'removeMe'.
  • Add a div of class 'target' into the div of class 'sandbox'.
  • Insert a paragraph in to the div you created.
  • Add text to the paragraph you created.
